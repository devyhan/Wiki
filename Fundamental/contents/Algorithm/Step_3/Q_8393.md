### 1.2.문제
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

### 1.3.입력
첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

### 1.4.출력
1부터 n까지 합을 출력한다.

### ~~1.5.제한~~

### 1.6.예제입력
*입력 1번*
```swift
3
```


### 1.7.예제출력
*출력 1번*
```swift
6
 ```


### 1.8.풀이
```swift
import Foundation

var n = Int(readLine()!)!
var sum:Int = 0

for n in 1...n {
    sum = sum + n
}

print(sum)
```
